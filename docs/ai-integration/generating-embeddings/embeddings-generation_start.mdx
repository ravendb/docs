# Generating embeddings: Start
[Embeddings](https://en.wikipedia.org/wiki/Embedding_(machine_learning)) are mathematical representations of content, where the content is converted into high-dimensional numerical vectors that capture semantic meanings and relationships.  
 - RavenDB allows you to create embeddings for document fields (e.g., texts, arrays, or metadata) using [ongoing embeddings-generation tasks](../../ai-integration/generating-embeddings/embeddings-generation-task), and use these embeddings for searches based on meaning rather than exact keyword matches.  
  - You can also create embeddings using external embeddings providers and place them in your database, e.g. to handle other content types like images.  
- You can apply [vector search](../vector-search/vector-search_start) operations to search over either pre-generated embeddings or embeddings generated on-the-fly while searching.  
- Pre-generated embeddings and vector search operations can be used by other RavenDB AI features. For example, [GenAI tasks](../../ai-integration/gen-ai-integration/gen-ai_start) can use vector search to find documents and update them with LLM-generated content, or [AI agents](../../ai-integration/ai-agents/ai-agents_start) can use vector search to retrieve relevant data requested by the LLM.

## Use cases
Embeddings generation tasks can be used to prepare your data for AI-powered search, analysis, and usage, e.g., for these data categories:

* **Enterprise knowledge bases**  
  Generate embeddings for thousands of documents, policies, and procedures to enable instant semantic search
* **Legal document libraries**  
  Process case law, contracts, and regulations to build searchable legal repositories
* **Product catalogs**  
  Convert product descriptions, specifications, and reviews into embeddings for enhanced e-commerce search
* **Content management systems**  
  Transform blog posts, articles, and marketing materials into searchable vector representations

See the [vector search](../../ai-integration/vector-search/vector-search_start#use-cases) use cases section for a few practical applications of embeddings in RavenDB.
