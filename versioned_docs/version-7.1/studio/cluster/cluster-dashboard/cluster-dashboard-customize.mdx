# Cluster Dashboard: Customize
<Admonition type="note" title="">

Customize the cluster dashboard according to your needs by:  

* **Removing** widgets you find redundant  
* **Adding** widgets that are useful for you  
* **Resizing** widgets  
* **Repositioning** widgets  

* In this page:  
  * [Remove Widget](../../../studio/cluster/cluster-dashboard/cluster-dashboard-customize.mdx#remove-widget)  
  * [Add Widget](../../../studio/cluster/cluster-dashboard/cluster-dashboard-customize.mdx#add-widget)  
  * [Resize Widget](../../../studio/cluster/cluster-dashboard/cluster-dashboard-customize.mdx#resize-widget)  
  * [Reposition Widget](../../../studio/cluster/cluster-dashboard/cluster-dashboard-customize.mdx#reposition-widget)  
</Admonition>
## Remove Widget

To remove a widget from the cluster dashboard -  

![Remove Widget](./assets/cluster-dashboard-04-remove-widget.png)

1. Hover over a widget's top bar to display the Recycle Bin button.  
2. Click the Recycle Bin button to remove the widget.  

    <Admonition type="info" title="">
     Removing a widget is **reversible**.  
     To return a widget that's been removed, [Add it](../../../studio/cluster/cluster-dashboard/cluster-dashboard-customize.mdx#add-widget) 
     to the cluster dashboard.  
    </Admonition>



## Add Widget

To add a widget to the cluster dashboard -  

1. **Open the Add Widget view**:  
   ![Add Widget button](./assets/cluster-dashboard-05_1-add-widget-control.png)

2. **Add widgets you find useful**:  
   ![Add Widget Window](./assets/cluster-dashboard-05_2-add-widget-window.png)
    * Widgets that can be added, are displayed in color.  
      ![Can Be Added](./assets/cluster-dashboard-05_3-add-widget-pickable.png)
    * Widgets that are already added to the dashboard, are grayed out and cannot be picked.  
      ![Cannot Be Added](./assets/cluster-dashboard-05_4-add-widget-unpickable.png)



## Resize Widget

To **maximize** a minimized widget -  
![Maximize Widget](./assets/cluster-dashboard-06-resize-widget-max.png)

1. Hover over the widget's top bar to display the Resize button.  
2. Click the Resize button to maximize the widget.  

To **minimize** a maximized widget -  
![Minimize Widget](./assets/cluster-dashboard-07-resize-widget-min.png)

1. Hover over the widget's top bar to display the Resize button.  
2. Click the Resize button to minimize the widget.  



## Reposition Widget

To reposition a widget, drag it to its new location.  
![Reposition Widget](./assets/cluster-dashboard-08-drag-widget.png)
