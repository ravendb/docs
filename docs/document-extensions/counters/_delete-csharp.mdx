## `Delete ` usage

**Flow**:  

* Open a session.  
* Create an instance of `CountersFor`.  
    * Either pass `CountersFor` an explicit document ID, -or-  
    * Pass it an [entity tracked by the session](../../client-api/session/loading-entities.mdx), 
      e.g. a document object returned from [session.Query](../../querying/overview.mdx) or from [session.Load](../../client-api/session/loading-entities.mdx#load).  
* Call `CountersFor.Delete`.
* Call `session.SaveChanges` for the changes to take effect.  

**Note**:

* A Counter you deleted will be removed only after the execution of `SaveChanges()`.  
* `Delete` will **not** generate an error if the Counter doesn't exist.
* Deleting a document deletes all its Counters as well.



## Example

<TabItem value="counters_region_Delete" label="counters_region_Delete">
<CodeBlock language="csharp">
{`// 1. Open a session
using (var session = docStore.OpenSession())
\{
    // 2. pass CountersFor's constructor a document ID  
    var documentCounters = session.CountersFor("products/1-C");

    // 3. Delete the "ProductLikes" Counter
    documentCounters.Delete("ProductLikes");

    // 4. Save changes to the session
    session.SaveChanges();
\}
`}
</CodeBlock>
</TabItem>
