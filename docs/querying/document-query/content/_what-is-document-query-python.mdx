<Admonition type="note" title="">

* In the RavenDB Python client, queries can be executed via `query`, `document_query`, or directly using raw `RQL`.  
  Learn more in [Query Overview](../../../querying/overview.mdx).      
    
* Unlike the **C#** client - where the `Query` method supports LINQ while `DocumentQuery` provides lower-level flexibility and control -
  the **Python** client API implements the `query` and `document_query` methods with identical functionality.
  Therefore, our Python documentation typically provides `query` samples without duplicating them for `document_query`.  

* In this article:
  * [`document_query` examples](../../../querying/document-query/what-is-document-query.mdx#document_query-examples)
  * [Available custom methods and extensions](../../../querying/document-query/what-is-document-query.mdx#available-custom-methods-and-extensions)

</Admonition>

<Panel heading="document_query examples">
    
#### Query collection - no filtering

<Tabs groupId='languageSyntax'>
<TabItem value="document_query" label="document_query">
```python
all_employees = list(session.advanced.document_query(object_type=Employee))
```
</TabItem>
<TabItem value="RQL" label="RQL">
```sql
from "Employees"
```
</TabItem>
</Tabs>

#### Query collection - by ID

<Tabs groupId='languageSyntax'>
<TabItem value="document_query" label="document_query">
```python
# Query collection by document ID
employee = (
    session.advanced.document_query(object_type=Employee)
    .where_equals("Id", "employees/1-A")
    .first()
)
```
</TabItem>
<TabItem value="RQL" label="RQL">
```sql
from "Employees" where id() == "employees/1-A"
```
</TabItem>
</Tabs>
    
#### Query collection - with filtering

<Tabs groupId='languageSyntax'>
<TabItem value="document_query" label="document_query">
```python
# Query collection - filter by document field
employees = list(
    session.advanced.document_query(object_type=Employee).where_equals("first_name", "Robert")
)
```
</TabItem>
<TabItem value="RQL" label="RQL">
```sql
from "Employees" where FirstName == "Robert"
```
</TabItem>
</Tabs> 
    
#### Query collection - with paging

<Tabs groupId='languageSyntax'>
<TabItem value="document_query" label="document_query">
```python
# Query collection - page results
products = list(session.advanced.document_query(object_type=Product).skip(5).take(10))
```
</TabItem>
<TabItem value="RQL" label="RQL">
```sql
from "Products" limit 5, 10 // skip 5, take 10
```
</TabItem>
</Tabs>
