# Activate License 

<Admonition type="note" title="">

* Each RavenDB cluster must be registered with its own license.  

* The license key is sent to the email address that was provided when obtaining the license.  

* You can activate your license using configuration keys, or via Studio.  

* In this article:
   * [Activate your license using a configuration key](../licensing/activate-license#activate-your-license-using-a-configuration-key)
   * [Activate your license via Studio](../licensing/activate-license#activate-your-license-via-studio)

</Admonition>

<Panel heading="Activate your license using a configuration key">

* You can activate your license using either of the following configuration keys:  
  * [License](../licensing/configuration.mdx#license)  
    Use this key to provide your license string directly, as the key's value.  
    E.g., `"License": "{ your license string }"`  
  * [License.Path](../licensing/configuration.mdx#licensepath)  
    Use this key to provide the path to a file containing your license.  
    E.g., `"License.Path": "D:\\RavenDB\\Server\\license.json"`  

* The configuration keys can be set in the `setting.json` file or as environment variables.  
  Learn more in the [configuration overview](../server/configuration/configuration-options.mdx) page.

</Panel>

<Panel heading="Activate your license via Studio">

To activate your license via Studio, open the **About** view and click the **Register license** option:

![Register License Button](./assets/register-license-button.png)

1. **About**  
   Click to open the **About** view.
   
2. **Register license**  
   Click to register and submit your license using the following form:  
   ![Register License Form](./assets/register-license-form.png)

</Panel>
