There are few methods that allow you to retrieve an index from a database:   
- [GetIndex](../../../../client-api/commands/indexes/get.mdx#getindex)   
- [GetIndexes](../../../../client-api/commands/indexes/get.mdx#getindexes)   
- [GetIndexNames](../../../../client-api/commands/indexes/get.mdx#getindexnames)   

## GetIndex

**GetIndex** is used to retrieve an index definition from a database.

### Syntax

<TabItem value="get_1_0" label="get_1_0">
<CodeBlock language="csharp">
{`IndexDefinition GetIndex(string name);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **name** | string | name of an index |

| Return Value | |
| ------------- | ----- |
| [IndexDefinition](../../../../glossary/index-definition.mdx) | Instance of IndexDefinition representing index. |

### Example

<TabItem value="get_1_1" label="get_1_1">
<CodeBlock language="csharp">
{`IndexDefinition index = store.DatabaseCommands.GetIndex("Orders/Totals");
`}
</CodeBlock>
</TabItem>



## GetIndexes

**GetIndexes** is used to retrieve multiple index definitions from a database.

### Syntax

<TabItem value="get_2_0" label="get_2_0">
<CodeBlock language="csharp">
{`IndexDefinition[] GetIndexes(int start, int pageSize);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **start** | string | Number of indexes that should be skipped |
| **pageSize** | int | Maximum number of indexes that will be retrieved  |

| Return Value | |
| ------------- | ----- |
| [IndexDefinition](../../../../glossary/index-definition.mdx) | Instance of IndexDefinition representing index. |

### Example

<TabItem value="get_2_1" label="get_2_1">
<CodeBlock language="csharp">
{`IndexDefinition[] indexes = store.DatabaseCommands.GetIndexes(0, 10);
`}
</CodeBlock>
</TabItem>



## GetIndexNames

**GetIndexNames** is used to retrieve multiple index names from a database.

### Syntax

<TabItem value="get_3_0" label="get_3_0">
<CodeBlock language="csharp">
{`string[] GetIndexNames(int start, int pageSize);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **start** | string | Number of index names that should be skipped |
| **pageSize** | int | Maximum number of index names that will be retrieved |

| Return Value | |
| ------------- | ----- |
| string[] | This methods returns an array of index **name** as a result. |

### Example

<TabItem value="get_3_1" label="get_3_1">
<CodeBlock language="csharp">
{`string[] indexNames = store.DatabaseCommands.GetIndexNames(0, 10);
`}
</CodeBlock>
</TabItem>
