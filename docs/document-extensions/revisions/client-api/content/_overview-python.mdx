## Revisions `store` Operations

* [ConfigureRevisionsOperation](../operations/configure-revisions.mdx) - apply a revision configuration
* [GetRevisionsOperation](../operations/get-revisions.mdx) - get revisions



## Revisions `session` methods

#### Get revisions:  

* Use [get_for](../session/loading.mdx#get-all-revisions) 
  to retrieve **all** revisions kept for a document.  
* Use [get_metadata_for](../session/loading.mdx#get-revisions-metadata) 
  to retrieve **metadata** for all revisions kept for a document.  
* Use [get_by_before_date ](../session/loading.mdx#get-revisions-by-creation-time) 
  to retrieve revisions by **change vector** or **creation time**.  
* Read [here](../../../../client-api/session/how-to/perform-operations-lazily.mdx#get-revisions) 
  about **lazy versions** of `get_for`, `get_metadata_for`, and `get_by_before_date`.  
#### Count revisions:  

Use [get_count_for](../session/counting.mdx#getcountfor) 
to get the number of revisions kept for a document.  
#### Force revision creation:

Read [here](../../overview.mdx#force-revision-creation-via-api) 
about creating a revision even if revision configuration is disabled.
