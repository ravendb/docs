<Admonition type="note" title="">

* In contrast to the .NET client,  
  the Node.js client provides a **unified API** for querying documents via the `session.query()` method.  

* The `query` method is essentially a shorthand for invoking the `documentQuery` method.  
  See examples of those equivalent calls in [query vs documentQuery](../../../../../client-api/session/querying/document-query/query-vs-document-query.mdx).

* All available methods for the session's `query` method are listed [here](../../../../../querying/overview.mdx#query-api).
