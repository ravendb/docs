# Configuration: Tombstone Options
<Admonition type="note" title="">

Use the configuration keys listed below to control RavenDB's tombstone cleanup.  

Learn how to apply these keys in the [Configuration Overview](../../server/configuration/configuration-options.mdx) article.

To inspect the current state of tombstone cleanup, open the [Tombstones view](../../studio/database/settings/tombstones.mdx) in Studio.

* On this page:
   * [Configuration keys](../../server/configuration/tombstone-configuration.mdx#configuration-keys)
      * [Tombstones.CleanupIntervalInMin](../../server/configuration/tombstone-configuration.mdx#tombstonescleanupintervalinmin)
      * [Tombstones.RetentionTimeWithReplicationHubInHrs](../../server/configuration/tombstone-configuration.mdx#tombstonesretentiontimewithreplicationhubinhrs)
      * [Tombstones.CleanupIntervalWithReplicationHubInMin](../../server/configuration/tombstone-configuration.mdx#tombstonescleanupintervalwithreplicationhubinmin)

</Admonition>

<Panel heading="Configuration keys">

<ContentFrame>

### Tombstones.CleanupIntervalInMin

Time (in minutes) between tombstone cleanups.

- **Type**: `TimeUnit.Minutes`
- **Default**: `5`
- **Scope**: Server-wide or per database

</ContentFrame>

<ContentFrame>

### Tombstones.RetentionTimeWithReplicationHubInHrs

Time (in hours) to save tombstones from deletion if this server is defined as a replication hub.

- **Type**: `TimeUnit.Hours`
- **Default**: `336` (14 days)
- **Scope**: Server-wide or per database

</ContentFrame>

<ContentFrame>

### Tombstones.CleanupIntervalWithReplicationHubInMin

Time (in minutes) between tombstone cleanups if this server is defined as a replication hub.

- **Type**: `TimeUnit.Minutes`
- **Default**: `1440` (1 day)
- **Scope**: Server-wide or per database

</ContentFrame>

</Panel>
